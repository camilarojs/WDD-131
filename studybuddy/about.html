About StudyBuddy

StudyBuddy was created to help students stay organized without feeling overwhelmed. It’s designed to be simple, easy, and fast, so you can focus on learning.

What It Does

  • Track tasks and assignments
  • Set study goals
  • Stay motivated with a clean, friendly layout

Why I Built It

I built this tool because managing deadlines can get stressful fast, especially with multiple classes. StudyBuddy helps keep everything in one place so planning feels more manageable.
