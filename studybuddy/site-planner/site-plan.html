Overview

Purpose

StudyBuddy is a study planner website that helps students stay organized by tracking assignments, deadlines, and study tasks in one web app. It aims to make planning simpler and reduce stress by giving users a clear overview of what they need to work on each day.

Audience

This site is designed for students, especially college and high school students, who want an easy, quick, and accessible tool to manage their school responsibilities. It helps them manage their study schedules and keep track of assignments in a simple way.

Dynamic Elements

  1. Task list using Objects and Arrays

    Each assignment will be stored as an object inside an array. This makes it easy to organize tasks and update them whenever the user adds or edits something.

  2. Array Method Usage

    I will use basic array methods to work with the task list:

    • filter() — show only certain tasks, such as upcoming or completed ones.
    • map() — prepare or transform the task data before displaying it.
    • forEach() — loop through all tasks and display them on the page.
  3. Functions to Manage Tasks

    I will write functions for adding tasks, marking them complete, deleting them, and sorting them by date. Each function will keep the task list updated and organized.

  4. DOM Interaction and Event Listeners

    JavaScript will respond to user actions such as clicking buttons or submitting forms. For example, when someone adds a task, the script will read the input and immediately display the new task on the screen.

  5. Conditional Logic

    I will use simple conditionals to check things like whether required fields are empty, whether a task is overdue, or whether a task needs a certain highlight based on its due date.

  6. ES Modules for better organization

    I will split the JavaScript into separate files and use ES modules to import and export functions or data. This keeps the code clean, modular, and easier to maintain.
